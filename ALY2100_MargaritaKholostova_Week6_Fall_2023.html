Assignment: Implement a Grade Calculator¶

Objective: This assignment aims to create a Python program that takes numeric grades as input and calculates the corresponding letter grade for each grade entered.

Instructions:

  1. Create a dictionary that maps numeric grade ranges to their corresponding letter grades. Use the following ranges and letter grades:

    • 90 and above: 'A'
    • 80 to 89: 'B'
    • 70 to 79: 'C'
    • 60 to 69: 'D'
    • Below 60: 'F'

  2. Write a Python program that repeatedly takes input from the user until they enter a negative value. Use the dictionary to determine the corresponding letter grade and print the result for each positive numeric grade entered.

  3. If the input grade is outside the range of 0 to 100, print an error message indicating that the input is invalid.

  4. After processing all the valid input grades, print a message displaying the total number of grades entered, the number of grades falling into each letter grade category, and the percentage of each letter grade category relative to the total number of grades entered.

# Explanation:
# This program is designed to take numeric grades as input and convert them to corresponding letter grades.
# The program will repeatedly take inputs until a negative value is entered.
# For each valid grade entered, it will output the letter grade and save this to a text file.
# The program will also save a final summary to the same file.

grades_dictionary = {'A': (90, 100), 'B': (80, 89), 'C': (70, 79), 'D': (60, 69), 'F': (0, 59)}

def get_grade(user_input):
    """Function to map numeric grade to its corresponding letter grade."""
    for grade, (min_score, max_score) in grades_dictionary.items():
        if min_score <= user_input <= max_score:
            return grade
    return None

grade_counts = {'A': 0, 'B': 0, 'C': 0, 'D': 0, 'F': 0, 'Invalid': 0}
total_grades = 0

# open a text file to save grades and summary.
with open("grades_summary.txt", "w") as file:

    while True:

        try:
            user_input = int(input("Enter a numeric grade (negative value to quit): "))

            # exit the loop for any negative value.
            if user_input < 0:
                break

            grade = get_grade(user_input)
            if grade:
                grade_counts[grade] += 1
                total_grades += 1
                message = f"Grade: {user_input} - Letter Grade: {grade}\n"
                print(message)
                file.write(message)
            else:
                error_message = "Error: Invalid grade input. Please enter a grade between 0 and 100.\n"
                print(error_message)
                file.write(error_message)
                grade_counts['Invalid'] += 1

        # for any input that is not a number.
        except ValueError:
            print("Error: Please enter a valid number.")
    
    file.write("\nGrade summary:\n")
    file.write(f"Total grades entered: {total_grades}\n")
    for grade, count in grade_counts.items():
        if grade != 'Invalid':
            message = f"Number of {grade}'s: {count} ({count/total_grades*100:.1f}%)\n"
            print(message)
            file.write(message)
    #only for input that is more then 100, when NONE is returned from get_grade funct
    if grade_counts['Invalid'] > 0:
        error_message = f"Number of invalid grades entered: {grade_counts['Invalid']}\n"
        print(error_message)
        file.write(error_message)
